# Blog-Backend-Data-Management-System
Blog Backend Data Management System is a Python-based desktop application built with SQLite and Tkinter featuring a modern dark GUI. It allows users to register, log in, create and manage blog posts, add comments, vote on posts. The system includes role-based access with an Admin Panel for managing users, posts, and comments.



##Blog Backend Data Management System

A Python + SQLite powered Blog Management System with a modern dark-themed Tkinter GUI designed for managing blog content efficiently. This application provides complete backend data handling with user authentication, post management, commenting, voting, and admin controls.

##Features

 User Registration & Secure Login (SHA-256 password hashing)
 Role-Based Access (Admin & User)
 Create, Edit, Delete & Search Blog Posts
 Categories & Tags Management
 Like / Dislike Voting System
 Post Commenting System
 Pagination & Search Filters
 User Profile Management
 Admin Panel for Users, Posts & Comments
 Dark Modern Tkinter Interface
